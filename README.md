# Winppy
Winppy는 Guppy C++ 게임 서버 라이브러리의 Windows 버전입니다.
![Winppy logo](images/logo.png)

## Features
Guppy 네트워크 라이브러리는 고성능 게임 서버 및 클라이언트 개발에 사용할 목적으로 개발되고 있으며, Windows 운영체제의 효율적인 입출력 처리모델인 I/O Completion Port를 이용하여 구현되어 있습니다.

주요 기능:
- 직렬화 버퍼 송신/수신

### Note
- 이 프로젝트는 x86, AMD64 및 Windows 7/10 플랫폼에서만 테스트되었습니다. 이 프로젝트의 중요 코어 부분들은 플랫폼 의존성이 존재하므로 다른 플랫폼에서는 정상적으로 작동하지 않을 수 있습니다.
